Twitter
twt
Goes to twitter.com.
twt [username]
Shows the user's profile.
twt -s [text]
Searches twitter for tweets containing the text.
twt -su [username] [text]
Searches for tweets from the given user containing the given text.
twt -sh [tag]
Searches twitter for tweets containing the given hashtag.
twt -sh [username] [tag]
Searches twitter for tweets from the given user containing the hashtag.
